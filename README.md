## ⚙️ Prerequisites

To run this project locally, make sure you have the following installed:

* **Node.js**: `v24.18.0` or higher

## 🛠️ Tech Stack

This website is built using the following technologies:

* **Frontend:** [Nuxt.js](https://nuxt.com/) (Vue 3 Framework) with **TypeScript** for type safety and robust development.
* **Backend:** [Express.js](https://expressjs.com/) (Node.js Framework) for handling APIs and server-side logic.

⚠️ Note: This project is currently under active development.
